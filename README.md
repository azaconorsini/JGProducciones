# JG Producciones R.L

**JG Producciones** es una empresa de servicios múltiples orientada a satisfacer necesidades y requerimientos de cualquier organización. Como diseñador y diagramador web mi función principal dentro de la compañía brindar presencia en internet a empresas que desean promover su marca a través de la web, elaborar diseños o bocetos y luego de ser aprovados por el cliente, llevaba a cabo el desarrollo de los proyectos.

## Responsabilidades

- Diseño de las interfaces de usuario, desde su comienzo hasta su correcta visualización en dispositivos móviles y de escritorio.
- Actualización y personalización de páginas web.
- Creación de páginas web empleando tecnologías como HTML5, CSS3, LESS, SASS, Bootstrap framework, WordPress y Javascript.
- Desarrollo de temas para WordPress.
